Pot-toolkit

Simple PHP scripts for conversion of Portable Text (POT) in/from CSV.

Download as .zip Download as .tar.gz View on GitHub

Gettext .po files toolkit

Here are useful commands about Gettext .po files.

Command pot2csv

Converts .po input file in Comma Separeted Value (_CSV_) format. Use help for more info:

$ ./potkit pot2csv --help

Command csv2pot

Converts .csv input file in Gettext .po format. Use help for more info:

$ ./potkit csv2pot --help
